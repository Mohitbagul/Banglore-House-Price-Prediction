# Banglore-House-Price-Prediction

Topic :- To predict the rate of a house in Banglore city for a region with different sizes, rooms, etc of a house.

Database :- Banglore city data of rates of houses/flats in particular areas.

Steps followed :-  
1) Data cleaning
2) Plotting the data to get insights for different algorithms.
3) Applying algorithms to predict data:-

    a) LinearRegression
   
    b) Decision Tree
4) Predicting price for a particular input.
